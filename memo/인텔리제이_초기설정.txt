▶ 유용한 IntelliJ 단축키
    Ctrl + D : 한 줄 복사
    Ctrl + X : 한 줄 삭제 및 잘라내기
    Shift + F6 : 이름 변경 (변수 메소드 등등)
    Ctrl + Alt + L : 자동 줄 맞춤
    Ctrl + Shift + F: 검색
    Ctrl + Shift + R: 바꾸기

▶ 초기 설정
    Auto import
        자동 임포트 옵션 활성화
        정확한 클래스가 하나만 있을 경우 자동으로 가져오기!
        자동으로 사용하지 않는 import문 제거!

    Code Completion
        대소문자를 구분할지 여부를 체크! 체크해제해야 대소문자 구분하지 않음!

    File Encoding
        인코딩 타입 결정! UTF-8 로 통일!

    Annotation processor
        주석 처리기! 사용 활성화!
