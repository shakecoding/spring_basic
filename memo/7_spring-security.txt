▷ Spring Security
    애플리케이션의 인증과 인가 기능을 제공하는 Spring 하위의 프레임워크

▶ 인증
    사용자의 계정을 확인하는 과정, 아이디와 비밀번호를 입력하여 인증 진행.
    ex) 기본 인증, 폼 인증, OAuth 등등

▶ 인가
    인증된 사용자가 특정 기능에 접근할 수 있는 권한을 확인하는 과정

▶ 필터 체인 (Filter Chain)
    - 여러 보안 필터가 체인 형태로 연결되어 요청을 처리한다.
    - 각 필터는 특정 보안 기능을 담당한다. (인증 필터, 인가 필터, 세션 관리 필터 등등)
